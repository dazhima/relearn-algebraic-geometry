Example 1.3

A priori, the Jacobson radical of a ring could be larger than the nilradical. Here is
an example of where this happens. Let (A, m) be a local ring which is an integral domain (e.g., the
p-adic numbers). Then N = (0) but R = m.
There is a way to characterize the elements of the Jacobson radical, analogous to the character-
ization of elements of the nilradical as nilpotent elements

Proposition 1.4

We have x ∈ R if and only if 1 − xy is a unit for every y ∈ A.

Proof.

Suppose that 1 − xy is not a unit for some y ∈ A. Then 1 − xy is contained in some maximal
ideal m of A. If x ∈ m then xy ∈ m and this would imply that (1 − xy) + xy ∈ m, which is a
contradiction. Hence x ∈/ m, and x ∈/ R.
For the other inclusion we prove the contrapositive. Suppose that x ∈/ R. Then there is a
maximal ideal m such that x ∈/ m. It follows that m + (x) = A, which means there exists y ∈ A
such that 1 − xy ∈ m, so 1 − xy is not a unit. (cid:3)

Radicals enjoy a few properties that are easy to prove and are left for you to work out:

(1) r (r(a)) = r(a).
(2) r(ab) = r(a ∩ b) = r(a) ∩ r(b).
(3) r(a) = A ⇐⇒ a = A.
(4) r(a + b) = r (r(a) + r(b)).

(5) if p is prime then r(pn) = p.

Definition 2.1

Two ideals a and b of a ring A are relatively prime if a + b = (1).
It is easy to see that if a and b are any two ideals of a ring A then

(a + b)(a ∩ b) ⊆ ab
